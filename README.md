# Podcaster: React + TypeScript + Vite

## Requirements

- npm and/or yarn

## Getting started

### Node

```bash
npm install
```

### Yarn

```bash
yarn install
```

## Launch app

### Node

```bash
npm run dev
```

### Yarn

```bash
yarn dev
```

## Launch test

### Node

```bash
npm run test
```

### Yarn

```bash
yarn test
```
